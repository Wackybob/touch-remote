Sencha Touch - JavaScript Framework

Welcome to Sencha Touch

Thank you for participating in the Beta for Sencha Touch. Sencha Touch is a JavaScript framework for developers creating applications targeted for touch interface devices such as the Apple iPhone, iPod touch and iPad, as well as any touch-enabled device in the Google Android family.
